Rejoindre Janmesh : Documentation

Janmesh - Rejoindre le mesh local

Pour démarrer

Sur votre machine ou via une meshbox

Sur votre machine

Si vous avez un minimum de connaissances techniques, vous pouvez configurer le routage mesh sur un ordinateur ordinaire. Janmesh utilise la pile réseau cjdns du projet Hyperboria qui fournit une allocation automatique d'adresse IPv6, un routage basée sur une hashtable distribuée (DHT), et un chiffrement point-à-point automatique. Cjdns fonctionne sous Linux (la configuration la plus simple à mettre en place), mais également MacOS, Windows, FirefoxOS, Android...
Pour plus d'information référez-vous à notre documentation

Via une meshbox

Une "meshbox" est un périphérique, qui peut être, par exemple, une ancienne tour, un ancien ordinateur portable ou netbook, ou un modem routeur même ancien, qui dispose de deux interfaces réseaux, l'une (souvent, du wifi) servant à se connecter à Janmesh, et l'autre étant connectée à votre ordinateur pour que vous accédiez aux services du mesh local sur celui-ci.
Si vous avez une machine ancienne, que vous voulez en faire une meshbox, vous trouverez plus d'information dans notre documentation.
Autrement, n'hésitez pas à nous contactez, si vous avez un ancien appareil que vous pouvez vouer à cette tâche, il y aura peut être un janmesher prêt à vous aider à y installer un système dédié pour en faire une meshbox, ou reconfigurer un ancien modem routeur dans ce but.
Également, vous pouvez consultez nos petites annonces pour voir s'il n'y n'a pas à vendre une machine préconfigurée en meshbox.

Vous avez le matériel configuré pour le mesh ? Et ensuite ?

Il y a ensuite plusieurs façons de rejoindre un mesh local.
La seule chose à savoir c'est qu'il faut assurer une connexion réseau avec une autre machine déjà sur le mesh.
Cette connexion peut être assurée via un tunnel au travers d'internet, une liaison wifi, une liaison CPL ou même une liaison filaire Ethernet voire (si vous avez les moyens) une liaison radio longue portée Wimax.

Tunnel via internet

Avant même d'être à portée de wifi, vous pouvez d'ore et déjà rejoindre le mesh local en mettant en place un tunnel au travers de votre connexion internet. La seule chose à faire est de trouver un pair ; pour celà, il suffit de nous contacter.

Wifi

Si vous pouvez voir dans votre liste de réseau wifi à portée un réseau dont le nom est http://janmesh.net, bravo, vous êtes à portée d'un des nœuds du réseau wifi Janmesh. Vous pouvez configurer une machine avec une interface dédiée à la connexion au mesh-local, ou monter (ou nous faire monter :) ) une meshbox, pour rejoindre le réseau.

CPL

La téchnologie CPL (courant porteur en ligne) permet de communiquer, avec des débits qui peuvent être très bon, entre machines situées sur le même réseau électrique que vous (connectées au même transformateur moyenne tension). Idéal pour rejoindre un noeud du mesh qui se trouve dans votre quartier mais est trop loin pour le wifi. Le principe reste le même, utiliser une meshbox ou configurer cjdns sur votre ordinateur. Cependant, il faut compter avec l'achat de deux interfaces CPLs pour faire la liaison entre vous et la machine qui voudra bien "peerer" Janmesh avec vous (comptez un peu moins de 40 euros).

Ethernet

Le meilleur moyen d'avoir un débit imbattable (jusqu'à 1000Mbps) avec un voisin immédiat est encore de tirer un cable ethernet entre les deux logements, pour connecter deux meshboxs ou ordinateurs équipés chacun d'une prise Ethernet dédié à établir cette liaison. Si vous êtes un peu bricoleur et que votre voisin fait partie du Janmesh, c'est peut-être la bonne solution pour vous.
