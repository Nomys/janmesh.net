Janmesh - Documentation Windows

Si vous voulez connecter une machine Windows à une meshbox configurée par quelqu'un d'autre :
  • Si vous connectez une meshbox directement à votre box internet/à votre réseau domestique, tous les périphériques de votre réseau local deviendront visible sur le Janmesh. Cela peux poser un problème avec, par exemple, les imprimantes d'usage courant, ou toute autre classe de périphériques qui n'incluent pas de controle des accès réseau. Il est plutôt conseillé de connecter la meshbox directement sur la machine cible sur laquelle vous voulez utiliser les application du réseau Janmesh
  • Si vous avez une meshbox sous Linux qui distribue localement une connectivité Janmesh directement à une machine Windows ou sur un réseau privé qui comporte des machines Windows, vous devrez sécuriser celles-ci selon la procédure indiquée dans la documentation (en anglais) du Projet Hyperboria. C'est également nécessaire pour les machines Linux, mais sous Windows c'est à la fois plus fastidieux et plus long.

    • Si vous voulez configurer un noeud Janmesh actif directement sur une machine Windows :

  • Windows ne supporte pas le routage olsrd ; vous devrez donc vous connecter avec un autre janmesher en liaison point à point uniquement
  • Il faut compiler sois même cjdns, à ce jour il n'y a pas de binaires précompilés de disponible
  • Les étapes de sécurisation de la machine sont plus complexes que sous Linux
    • Dans tous les cas, nous vous renvoyons aux sections consacrées à Windows de la documentation du Projet Hyperboria
